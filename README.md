# 🎬 newTube – YouTube Backend Style Project

A backend-focused YouTube clone built using JavaScript.  
This project is part of a backend learning series and demonstrates how a video platform server can be structured using modern backend practices.

---

## 🚀 About The Project

**newTube** is a backend-driven application inspired by YouTube’s core architecture.  
The main goal of this project is to understand backend development concepts such as API design, authentication, database handling, and scalable server structure.

This repository focuses mainly on:

- REST API development
- Backend architecture
- Environment configuration
- Project structuring for real-world apps

---

## ⚙️ Features

✅ User authentication & authorization  
✅ Video upload structure (backend logic)  
✅ API-based architecture  
✅ Environment variable configuration  
✅ Scalable folder structure  
✅ Clean code formatting using Prettier  

---

## 🧰 Tech Stack

- **JavaScript (Node.js)**
- **Express.js** (Backend Framework)
- **REST APIs**
- **Environment Variables (.env)**
- **Prettier** for code formatting
